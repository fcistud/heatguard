Glossary

Key terms

The vocabulary used across the HeatGuard docs, in one place.

WBGT
Wet-Bulb Globe Temperature; the standard occupational heat-stress index (ISO 7243), combining humidity, radiant/sun load, and air temperature. See how it's computed.
PHS
Predicted Heat Strain (ISO 7933); a physiological model of core temperature and sweat loss over time. HeatGuard uses it for the hydration target and the max-exposure cap.
TLV / Action Limit
ACGIH screening thresholds for acclimatized (TLV) / unacclimatized (Action Limit) workers. The Action Limit is ~3 °C stricter throughout. See the work-rest cycle.
Acclimatization
The body's ~1–2-week adaptation to heat; new arrivals are most at risk. HeatGuard enforces a NIOSH-based exposure ramp for them.
AKI
Acute Kidney Injury; the kidney damage caused by repeated heat-and-dehydration exposure during manual labour. The headline harm HeatGuard is designed to prevent.
Metabolic category
The work intensity (light / moderate / heavy / very-heavy), per ISO 8996. Drives the work-rest table lookup and the PHS metabolic rate (1 met = 58.15 W/m²).
The calendar ban
The Gulf's fixed midday outdoor-work prohibition — a clock window on fixed dates — that HeatGuard replaces. See why it is wrong in both directions.
The signal
The single hourly broadcast: WORK / REST IN SHADE / DRINK NOW / STOP. One cheap, universally understood instruction for the whole site.
Liljegren model
The validated 2008 algorithm (via ECMWF's thermofeel) HeatGuard uses to estimate outdoor WBGT from ordinary weather plus the sun's position.
Provenance
The source tag every WBGT value carries — liljegren, fallback, or measured — so the approximation is never hidden in the log or the UI.
Compliance log
The append-only, SHA-256 hash-chained record of readings, breaks, drink prompts, and STOPs. Worker-protective first, a compliance shield second, and privacy by design.
Adelante Initiative
The La Isla Network's Water-Rest-Shade program (termed RSHH / RSH-S) at Ingenio San Antonio, Nicaragua — the source of the AKI −94% and productivity +10–20% effect sizes HeatGuard's impact model rests on.
